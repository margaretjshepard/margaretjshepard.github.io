TraffiQ: Live Traffic Intelligence Dashboard

Mar 2025 – Apr 2025 · William & Mary

TraffiQ is a real‑time traffic intelligence dashboard that leverages live congestion data to help organizations make smarter decisions about road usage and transportation. The goal of the project was to show how businesses could use streaming data to support route planning, time management and operational decision‑making.

We collected traffic data from twenty manually defined zones across four U.S. cities using the TomTom Traffic API, refreshing the dataset every 30 seconds【355415436035554†screenshot】. To handle the streaming data, we integrated Google Pub/Sub for message handling and wrote a backend in Python to parse the API responses and standardize congestion metrics on a 0–1 scale. On the front end, we built an interactive dashboard in Plotly Dash where users can select a city and explore heat maps and dynamic visualizations of congestion levels in real time. This architecture allowed us to deploy a responsive web app that updates continuously as new data arrives【355415436035554†screenshot】.

During the presentation we demoed the live dashboard, discussed challenges such as manual geolocation and inconsistent metrics, and proposed improvements like user‑experience refinements, better reliability and predictive modelling. The project demonstrated the potential of combining streaming APIs, cloud message queues and modern Python visualisation frameworks to build decision‑support tools.

Technologies used

  • Python & Plotly Dash for the interactive dashboard
  • Google Pub/Sub for streaming data ingestion
  • TomTom Traffic API for real‑time congestion data
  • Mapbox for base maps and heat map visualisations
